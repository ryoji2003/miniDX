#miniDX Project
# logic
## role
**役割に"(数字)"となっているのは、その業務に必要な人数**
相談(1)
看護師(2)-------訓練(1)  
         |
        　-----看護(1)
特浴(1)   
風呂(5)
リーダー(1)
サブリーダー(1)
運転(more than 6)

# Constraints
-"看護"は絶対に"看護師"の人が行う。
-"訓練"は訓練と運転のみが可能で他の業務ができない者がいる。
-"リーダー"は"相談"、"相談員"、"看護師"、"介護"中から一人選ばれる。
-運転は6人以上で行う。
-毎日11人で業務を行う。
-パートの人が途中で抜けても、その人数でその日は業務を行う。
## contents
-ワゴン車が4台、普通車が2台ある。
-運転できる車種が限定されている。(普通車は運転できるがワゴン車は運転できる人や、運転できない人)
-パート職員は短時間勤務であり、送迎業務には就かない。
